环境
----

* 安装JDK6
* 安装Maven3

启动
----

* 执行reset.bat初始化数据库
* 执行start.bat启动应用

使用
----

* 访问http://localhost:8080/activiti-demo
* 点击“首页”->“启动流程”开始体验
